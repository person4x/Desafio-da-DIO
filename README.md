Desafio Yu-Gi-Oh Jo-Ken-Po
Este projeto foi desenvolvido como parte do desafio da DIO, combinando o universo de Yu-Gi-Oh com a mecânica clássica do jogo Jo-Ken-Po (pedra, papel e tesoura). O objetivo é proporcionar uma experiência divertida e interativa com personagens e cartas inspiradas no anime.
Funcionalidades
Escolha entre cartas de monstros com diferentes atributos
Sistema de combate baseado em Jo-Ken-Po
Efeitos visuais e sonoros para tornar a batalha mais imersiva
Interface responsiva e estilizada com CSS

Tecnologias utilizadas
HTML5, CSS3, JavaScript, Git e GitHub
